K歌自助更新CK

快速更新全民K歌 Cookie · 多端教程

🦸 安卓 🍎 苹果 💻 电脑
手机 Cookie 获取方法（无需电脑，两个手机也行）
  1. 手机下载 Alook 浏览器
  2. 打开 全民K歌官网
  3. 浏览器菜单 → 电脑版（请求桌面版网页方便扫码）
  4. 右上角登录 → 弹出二维码 → 截屏二维码到另一个设备
  5. 打开全民K歌 APP 扫码登录
  6. 登录成功后，浏览器菜单 → 工具箱 → 开发者工具 → Cookies → 拷贝
  7. 粘贴到上方输入框，点击"更新 CK"即可
▶ 视频教程
苹果也可用安卓方法，以下为 Stream 抓包方案
  1. 下载 Stream 抓包软件
  2. 打开 Stream → 设置 → HTTPS 抓包 → 安装 CA 证书 → 允许
  3. 跳转到浏览器下载证书（需连 WiFi）
  4. 手机设置 → 通用 → 描述文件 → 安装证书
  5. 手机设置 → 通用 → 关于本机 → 证书信任设置 → 启用证书
  6. 返回 Stream → 设置 → HTTPS 抓包 → 确认 CA 证书已安装且信任
  7. 点击开始抓包 → 去全民K歌浏览"K歌账户" → 返回 Stream 停止抓包
  8. 抓包历史 → 第一条 session → 按域名找 node.kg.qq.com
  9. 任选一条记录 → 请求 → 请求头部 → 复制 Cookie
  10. 粘贴到上方输入框，点击"更新 CK"即可
▶ 视频教程
以 QQ 浏览器为例，其他浏览器大同小异
  1. 浏览器打开 全民K歌官网
  2. 右上角登录 → 弹出二维码 → 打开全民K歌 APP 扫码登录
  3. 按 F12 进入控制台 → 按 F5 刷新网页
  4. 按 图片指引依次进入 Network → Doc → index.html 或 index_pc.html
  5. Headers → 找到 Cookie → 复制到上方输入框
  6. 点击"更新 CK"即可
Cookie 数据仅用于维持登录状态
